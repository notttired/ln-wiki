# Call of the Dead

| | |
|---|---|
| **Type** | Active Skill |
| **User** | Death Fiend |
| **Properties** | |
| Summoned Unit | Ghoul |
| Number Summoned | 10 |
| Effect Trigger | Tremors |

Call of the Dead is an active skill utilized by the Death Fiend, a powerful undead entity. When activated, the skill causes the ground to tremble before ten Ghouls emerge from beneath the earth to assist the Death Fiend in combat. The skill was demonstrated during a confrontation, where the summoned Ghouls served as additional forces against the enemy.

## Effect

Upon activation, Call of the Dead induces localized tremors, signaling the imminent emergence of exactly ten Ghouls. These summoned creatures rise from the ground and fight alongside the Death Fiend, bolstering its offensive capabilities.
