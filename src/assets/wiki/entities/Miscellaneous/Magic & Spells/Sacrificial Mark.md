# Sacrificial Mark

| | |
|---|---|
| **Skill Name** | Sacrificial Mark |
| **Original User** | Vampire Duke Cambormir |
| **Current User** | Bjorn |
| **Source** | Vampire essence |
| **Properties** | |
| Range | 100 meters |
| Effect | Temporarily increases physical abilities per marked target |
| Activation Condition | Contact with user's acidic blood |

The Sacrificial Mark is a buff skill originally belonging to Vampire Duke Cambormir. It allows its user to mark targets within a 100-meter radius by applying a glowing red tattoo on their forehead via contact with the user's acidic blood. For each marked target, the user's physical abilities temporarily increase, making the skill more potent when employed against groups. Since acquiring the Vampire essence, Bjorn has used the Sacrificial Mark to great effect, notably during the prison riot where he marked dozens of inmates to boost his strength and speed. The skill's activation condition requires the user's blood to make contact with the target, meaning the user must already be wounded or deliberately draw blood to apply the mark. This limitation makes it a risky but high-reward ability in close-quarters combat.

## Details

The Sacrificial Mark is a skill that operates by marking targets within a 100-meter radius. The mark appears as a glowing red tattoo on the target's forehead and is applied when the user's acidic blood makes contact with the target. For each marked target, the user's physical abilities are temporarily enhanced, scaling with the number of sacrifices. The skill was originally used by Vampire Duke Cambormir and later acquired by Bjorn through the Vampire essence. Its reliance on the user's blood makes it a dangerous but potentially devastating ability in crowd situations.
