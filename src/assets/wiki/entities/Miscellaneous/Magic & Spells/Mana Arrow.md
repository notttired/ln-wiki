# Mana Arrow

| | |
|---|---|
| **Type** | Attack Spell |
| **Caster** | Raven |
| **Properties** | |
| Casting Time | Instant |
| Effect | Fires an arrow of light |

Mana Arrow is a basic attack spell used by Raven. It generates an arrow of light that can be fired instantly, requiring no casting time. This efficiency makes it a staple tool in Raven's repertoire, allowing for rapid and precise strikes.

The spell's most notable application occurred during an encounter with a Banshee. Raven used Mana Arrow not to inflict damage, but to mark the creature, thereby triggering the Mark of Grudge. This demonstrates the spell's versatility beyond mere offense.

As a foundational spell, Mana Arrow is characterized by its low mana consumption and quick activation. Its light-based nature suggests an elemental affinity, though its primary function remains straightforward magical damage. The spell serves as a reliable option for both combat and utility purposes.
