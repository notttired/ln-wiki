# Smash

| | |
|---|---|
| **Type** | Active Skill |
| **MP Cost** | 0 |
| **Attack Power** | Decent |

Smash is a combat skill employed by Bjorn that allows him to strike enemies with either his shield or a hammer. The skill delivers decent attack power and consumes no MP, making it a reliable and sustainable option in battle. Its versatility enables Bjorn to adapt to different combat scenarios, using the shield for defensive strikes or a hammer for heavier damage.

When used with a proper weapon, such as a two-handed hammer, Smash's effectiveness increases significantly. In Chapter 8, Bjorn demonstrates this by defeating goblins with the skill while equipped with a newly acquired hammer, showcasing the enhanced power that a suitable weapon provides.

## Overview

Smash is an active skill used by Bjorn that can be executed with a shield or a hammer. It has no MP cost and provides decent attack power, making it a versatile and efficient offensive tool.

## Usage

Bjorn employs Smash with both his shield and hammer. While the shield version offers moderate power, using a proper weapon like a two-handed hammer amplifies the skill's damage. In Chapter 8, Bjorn uses Smash with his new hammer to quickly dispatch goblins, demonstrating the skill's scalability with weapon quality.
